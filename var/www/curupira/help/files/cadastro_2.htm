Usuários

Permite cadastrar as informações sobre os usuários do sistema. O nível 2 pode cadastrar novos usuários, modificar a quota de um usuário, modificar o perfil de acesso e recolocar a quota do usuário.

Cadastramento

Os usuários são cadastrados automaticamente na primeira impressão feita no servidor.

Modificar Informações do usuário

Deve-se procurar pelo usuário a ser editado e clicar sobre o ícone de edifício ( à esquerda sob a coluna Ação). A página é preenchida com as informações do usuário. Os três passos a seguir podem ser feitos após esta Ação. Para gravar qualquer alteração nos dados do usuário basta clicar em gravar.

Para atualizar as quotas do usuário basta clicar modificar as quotas no Campo Permições e salvar clicando no botão Modificar (deixar o campo quota em branco caso nível se deseje estabelecer limite).

Para recolocar as quotas deve-se clicar sobre o botão Recoloca quota.
